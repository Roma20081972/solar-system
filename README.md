# 🌞 Интерактивный симулятор Солнечной системы

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue)](https://www.python.org/)
[![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)](https://docs.python.org/3/library/tkinter.html)
[![Pillow](https://img.shields.io/badge/PIL-Pillow-orange)](https://python-pillow.org/)

Великолепный интерактивный симулятор Солнечной системы с реалистичным движением планет, подробной информацией и удобным управлением.

![Скриншот программы](screenshot.png)

## ✨ Возможности

### 🪐 Визуализация
- Реалистичное движение 8 планет вокруг Солнца
- Кольца Сатурна с эффектом полупрозрачности
- Мерцающие звезды на заднем фоне
- Солнце с эффектом короны и лучей
- Орбиты планет с отображением расстояния в AU

### 📊 Информация о планетах
При наведении курсора на планету отображается подробная информация:
- 📏 Размеры и масса
- 🌡️ Температура поверхности
- 🪐 Расстояние от Солнца
- ⏳ Продолжительность года и суток
- 🌪️ Состав атмосферы
- 🛰️ Количество спутников
- ✨ Интересные факты

### 🎮 Управление
- **Мышь**: перемещение по карте (перетаскивание)
- **Колесико**: масштабирование
- **Наведение**: информация о планете
- **Панель управления**: скорость и масштаб
- **Кнопки**: пауза, сброс, центрирование

### ⌨️ Горячие клавиши
- `Пробел` - пауза/возобновление
- `+ / -` - увеличение/уменьшение масштаба
- `R` - сброс вида
- `F` - показать всю систему
- `O` - включить/выключить орбиты
- `L` - включить/выключить метки планет

### 🔍 Дополнительные функции
- Поиск планет по названию
- Отображение текущего времени в годах
- Индикация положения планеты на орбите
- Регулировка скорости анимации (от 0.1x до 100x)
- Строка состояния с текущими параметрами

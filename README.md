### 🎮 СИСТЕМА УПРАВЛЕНИЯ

#### **МЫШЬ И КЛАВИАТУРА**

| Действие | Управление | Описание |
|----------|------------|----------|
| Перемещение по карте | Зажать ЛКМ + двигать | Плавный pan по космосу |
| Масштабирование | Колесико мыши | Приближение/отдаление |
| Информация о планете | Навести курсор | Мгновенное отображение данных |
| Пауза | Пробел | Заморозка движения планет |
| Увеличение масштаба | `+` или `=` | Приблизить вид |
| Уменьшение масштаба | `-` | Отдалить вид |
| Показать всю систему | `F` | Автоподбор масштаба |
| Сброс вида | `R` | Вернуться к начальным настройкам |
| Вкл/Выкл орбиты | `O` | Показать/скрыть траектории |
| Вкл/Выкл метки | `L` | Показать/скрыть названия |

### 🔧 ПОШАГОВАЯ УСТАНОВКА

#### **ШАГ 1: Установка Python**

1. Перейдите на [официальный сайт Python](https://www.python.org/downloads/)
2. Скачайте последнюю версию Python 3.8 или выше
3. Запустите установщик
4. **ВАЖНО!** Отметьте галочку "Add Python to PATH"
5. Нажмите "Install Now"

#### **ШАГ 2: Проверка установки Python**
